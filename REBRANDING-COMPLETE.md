## 🎨 DESIGN SYSTEM

### **Trupeer Purple Palette**

```css
/* Primary Colors */
--trupeer-purple: #6C5DD3;      /* Main brand color */
--trupeer-purple-hover: #5B4EC2; /* Hover state */
--trupeer-purple-active: #4A3FB1; /* Active/pressed state */

/* Backgrounds */
--trupeer-purple-light: #F5F4FF; /* Light purple background */
--trupeer-purple-border: #E4E1FF; /* Purple borders */

/* Usage Examples */
.button-primary {
  background: var(--trupeer-purple);
}
.button-primary:hover {
  background: var(--trupeer-purple-hover);
}
.button-primary:active {
  background: var(--trupeer-purple-active);
}
.highlight-bg {
  background: var(--trupeer-purple-light);
  border: 1px solid var(--trupeer-purple-border);
}
```
